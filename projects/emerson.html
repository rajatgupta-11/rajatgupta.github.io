Conservative ROI (Assumptions)

  • Shift: 1 per day, 250 working days/year
  • Units/shift: 29 (observed). Subprocess delta: 13.44 s/unit
  • Labor burden: $22/hour (operator + overhead)

Labor savings (subprocess only)

  • Time saved/shift = 29 × 13.44 s = 389.76 s ≈ 6.5 min
  • Annual time saved = 6.5 min × 250 = 27.1 hours
  • Annual labor $ = 27.1 × $22 ≈ $596 per cell (subprocess slice)

Note: This is a floor estimate on a single measured segment. When upstream/downstream are aligned, realized gains scale with bottleneck removal and batch sequencing.

Representative Fixes

  • Batch/group similar motions → fewer tool swaps; lower fatigue
  • Guided alignment features for heavy parts → faster set
  • Standardize table heights → fewer crane cycles
  • Power tools for repetitive clamps → consistent timing
  • Stagger testing to overlap waits → cut idle time
