Problem

Non-standardized workflow and NVA motions increased lead time, fatigue, and costs; need a systematic method to expose bottlenecks and standardize processes across cells. :contentReference[oaicite:3]{index=3}

Approach: Implement MOST to quantify motion/time, remove NVA, reduce cycle time, and create a replicable model. :contentReference[oaicite:4]{index=4}

Objectives

  • Identify and eliminate bottlenecks; group similar activities to reduce motion/fatigue. :contentReference[oaicite:5]{index=5}
  • Optimize testing phase to minimize idle time. :contentReference[oaicite:6]{index=6}
  • Measure productivity improvements; standardize for replication across cells. :contentReference[oaicite:7]{index=7}

Methodology

  1. Study cell layout & current flow (Cell 2). :contentReference[oaicite:8]{index=8}
  2. Time Study: stopwatch + motion breakdown. :contentReference[oaicite:9]{index=9}
  3. MOST: encode motions (TMU), compute time. (1 TMU = 0.036 s). :contentReference[oaicite:10]{index=10}
  4. PERT: visualize tasks and identify critical waits. :contentReference[oaicite:11]{index=11}

MOST sequences: General Move, Controlled Move, Tool Use; assign indices → TMU → seconds. :contentReference[oaicite:12]{index=12} :contentReference[oaicite:13]{index=13}

Results (subprocess)

  • Time Study: 92 s (baseline). :contentReference[oaicite:14]{index=14}
  • MOST: 79.56 s (TMU 2210 × 0.036 s). :contentReference[oaicite:15]{index=15}
  • Δ = −13.44 s per unit; exposes inefficiencies & motion waste. :contentReference[oaicite:16]{index=16}

Key Bottlenecks → Fixes

  • Testing waits → Stagger batches to overlap test cycles, reduce idle time. :contentReference[oaicite:17]{index=17}
  • Task grouping → Batch screwing then power-fasten; cut repetition/fatigue. :contentReference[oaicite:18]{index=18}
  • Alignment time → Add detachable guiding protrusions for heavy parts. :contentReference[oaicite:19]{index=19}
  • Height mismatches → Standardize table heights; reduce crane moves. :contentReference[oaicite:20]{index=20}
  • Manual clamp → Use powered tool to cut clamp time. :contentReference[oaicite:21]{index=21}
