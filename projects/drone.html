Vision-Guided Payload Delivery UAV

Objective: track target with onboard vision, hold aim, and drop payload with CEP ≤ 15 cm @ ≤ 3 m altitude.

Acceptance Criteria

  • Hover ≥ 30 s (alt hold/pos hold)
  • Vision lock ≤ 1 s
  • Servo drop latency ≤ 0.2 s
  • CEP ≤ 15 cm @ 3 m (10 drops)
  • Failsafe: auto-land on signal loss
  • Mission endurance ≥ 3 min
  • Code + CAD + wiring in repo
  • Indoor + outdoor tests
